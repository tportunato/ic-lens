# IC Lens — PE Deal Intelligence

A private equity deal comparison dashboard for analysing 3–5 buyout deals across six analytical dimensions with return attribution waterfall, assumption aggressiveness scoring, and AI synthesis.

## Deploy to Vercel

1. Extract this zip
2. Run `npm install`
3. Run `npm run dev` to preview locally
4. Push to GitHub and import into Vercel, or use `npx vercel` to deploy directly

## Stack
- React 18 + Vite
- No external UI libraries
- DM Sans (Google Fonts)
